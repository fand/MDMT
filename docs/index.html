VEDA

VJ system on Atom

VEDA is an Atom package for VJ / Livecoding with GLSL.

Easy GLSL runtime

VEDA supports both fragment shaders and vertex shaders. In fragment shaders, you can use uniform variables like GLSL Sandbox. In vertex shaders, you can use uniform variables like vertexshaderart.com.

Cross-platform by design

VEDA is built on web technologies and provided as an Atom package. It works on Windows, macOS, and Linux.

Auto-completion, Linters, ...

Because VEDA runs on Atom, you can write GLSL with your favorite settings, package, and snippets. Also, VEDA automatically installs autocomplete-glsl and linter-glsl for rapid GLSL development.

Audio, MIDI, OSC, and more!

VEDA supports a variety of inputs such as:

  • Audio
  • MIDI
  • OSC
  • WebCam
  • Gamepad
  • Keyboard
  • Mouse
